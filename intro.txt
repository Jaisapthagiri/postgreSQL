DATABASE - A database is a structured collection of data stored on a computer. It is a powerful tool for storing and organizing data efficiently.

SQL - language used to control and query databases

RDBMS -  This is the most common database type, organizing data into tables with rows and columns. The popular RDBMS are PostgreSQL, MySQL, MariaDB, Oracle Database, SQL Server, and IBM Db2.

Data Types - String, Number , Boolean, Date, Time, TimeStamp

Operators - = , < , >, <=, >=, <>(!=), Like, ILike, AND, Or, Not, In, Between, Is Null

DDL - Create, Alter, Drop, Trucate 

DML - Select, insert, update, delete

Aggregate - Sum, Count, Avg, Min, Max

Numbers → INT, NUMERIC
Text → TEXT, VARCHAR
True/False → BOOLEAN
Date → DATE, TIMESTAMP
Advanced → JSONB, ARRAY
